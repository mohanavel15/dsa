# Linked List Problems

Linked list data structure and related problems.

## Problems

1. **Singly Linked List** (`singly_linked_list.py`)
   - Basic linked list implementation with operations
   - Insert, delete, search operations

2. **Reverse Linked List** (`reverse_linked_list.py`)
   - Reverse a singly linked list
   - Time: O(n), Space: O(1)

## Key Concepts

- Node structure
- Pointer manipulation
- Two pointer technique
- Fast and slow pointers
- Cycle detection
