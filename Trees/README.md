# Tree Problems

Binary tree and tree-based data structures.

## Problems

1. **Binary Tree Traversal** (`binary_tree_traversal.py`)
   - Inorder, preorder, postorder, level order traversals
   - Time: O(n), Space: O(h)

2. **Maximum Depth** (`maximum_depth.py`)
   - Find maximum depth of binary tree
   - Time: O(n), Space: O(h)

## Key Concepts

- Tree traversals (DFS, BFS)
- Recursive tree algorithms
- Binary Search Trees
- Tree properties (height, depth, balance)
