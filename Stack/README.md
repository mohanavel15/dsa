# Stack Problems

Stack data structure and applications.

## Problems

1. **Stack Implementation** (`stack_implementation.py`)
   - Basic stack implementation using list
   - Push, pop, peek operations

2. **Valid Parentheses** (`valid_parentheses.py`)
   - Check if parentheses are balanced
   - Time: O(n), Space: O(n)

## Key Concepts

- LIFO (Last In First Out) principle
- Expression evaluation
- Balanced parentheses
- Monotonic stack
