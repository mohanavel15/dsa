# Array Problems

Array-based data structure problems and solutions.

## Problems

1. **Two Sum** (`two_sum.py`)
   - Find two numbers that add up to a target
   - Time: O(n), Space: O(n)

2. **Maximum Subarray** (`maximum_subarray.py`)
   - Find contiguous subarray with largest sum (Kadane's Algorithm)
   - Time: O(n), Space: O(1)

## Key Concepts

- Hash maps for O(1) lookup
- Two pointer technique
- Sliding window
- Dynamic programming on arrays
